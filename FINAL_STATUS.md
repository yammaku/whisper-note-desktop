### 3. ✅ 修复文件夹路径

## 工作流程

1. **音频处理**:
   - 压缩为 32kbps MP3
   - 使用 11Labs API 转录
   - 备份原始文件和转录文本

2. **笔记更新**:
   - 新内容插入到标题后面
   - 最早的音频在最上面
   - 使用 #### 分隔每条笔记

3. **同步时间**:
   - 等待45秒确保iCloud同步

## 使用方式

### 命令行:
```bash
./voice_to_notes.sh audio1.wav audio2.wav
```

### Web界面:
1. 访问 http://localhost:8181
2. 拖拽音频文件
3. 点击"开始处理"
